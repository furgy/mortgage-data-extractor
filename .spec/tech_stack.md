# Tech Stack

## Language & Runtime
-   **Python 3.12+**: Core logic and scripting.
-   **uv**: Package management and environment isolation.

## Libraries
-   **pypdf**: For extracting text from mortgage PDF statements.
-   **csv**: Standard library for processing Property Boss and Stessa files.
-   **datetime**: For sorting and date manipulation.
-   **re**: For pattern-based data extraction.

## Development Tools
-   **Git**: Version control (Remote: GitHub).
-   **OpenSpec**: Project specification and documentation standard.
-   **Antigravity**: AI Pair Programming assistant.
